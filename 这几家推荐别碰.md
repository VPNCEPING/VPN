## 翻墙软件 / 技术黑名单（高风险）
> 说明（安全从业者视角）：下面这些名称在中文互联网里长期被大量“广告投放 / 捆绑下载 / 诱导付费 / 伪装成免费工具”的套路反复利用。  
> 我无法对每个软件当前版本逐一做源代码级审计，但从风险控制角度，它们更容易出现：钓鱼、隐私泄露、强制注入广告、安装额外组件、卷款跑路、证书劫持等问题。  
> 结论：**不建议在主力手机/电脑上安装**；如必须测试，请用备用机或虚拟机隔离。

## 安全从业者：如何下载软件与查毒自检
### 1）只从“可信来源”下载（优先级从高到低）
- **GitHub 官方仓库的 Releases 页面**（看作者、Star、提交历史、Issue 质量）
- **官网 + 可验证签名/校验值（SHA256）**（能校验再下）
- **正规应用商店**（iOS App Store / Google Play）
> 安卓第三方商店、网盘链接、QQ群文件、论坛附件：风险显著更高。
### 2）下载后先做“文件完整性校验”（能做就做）
如果对方提供了 `SHA256`：
- **Windows（PowerShell）**
  ```powershell
  Get-FileHash .\xxx.exe -Algorithm SHA256
- **MacOS / Linux**
<BASH>
  ```shasum -a 256 xxx.dmg
把你算出来的值与官网 / GitHub 提供的值对比，一致再安装。
没有校验值不等于一定有毒，但等于你无法确认是否被中间人篡改/二次打包。

## 3）至少做 2 次“多引擎查毒”
- **VirusTotal（多引擎扫描）**：上传安装包或粘贴下载链接  
  - 重点看：是否大量引擎报毒、是否命中 Trojan/Spyware/Adware、社区评论  
  - 小技巧：如果只有 1–2 家冷门引擎报 “Generic”，可能是误报；如果十几家以上一致报毒，基本不用犹豫。
- **本机杀软二次扫描**
  - Windows Defender（离线扫描更强）
  - macOS：可用 XProtect（或补充其他安全软件）

## 4）安装时的“危险信号”清单（出现就停）
- 要求 **通讯录 / 短信 / 无障碍权限 / 设备管理权限**（尤其 VPN/浏览器类）
- 强制安装 **证书 / 描述文件**（iOS 特别危险：可能导致 HTTPS 被中间人解密）
- 安装过程中捆绑一堆“加速器/浏览器/游戏助手”
- 首次启动就弹出“登录账号/绑手机号/开会员才可用”，且无透明公司信息

## 5）用“隔离环境”先跑一遍（推荐）
- Windows：虚拟机（VMware/VirtualBox）或 Windows Sandbox（部分版本提供）
- Android：备用机 / 工作资料分身（至少别用主力号）
- 不要在主力电脑上直接导入订阅链接/粘贴配置（配置里可能包含追踪域名）

## 6）网络侧自检：它到底在连谁？（进阶但有效）
至少做一个：
- Windows：资源监视器 / TCPView 看进程在连接哪些域名/IP
- macOS：活动监视器 + `lsof`（或 Little Snitch 这类防火墙）
- 路由器侧：看 DNS 请求是否异常、是否频繁连接奇怪域名

###（可选）最低安全基线（个人建议）
- 不碰来历不明的“国产魔改浏览器 + 一键翻墙”组合
- 优先选开源客户端（例如 Clash / sing-box 系）+ 相对可信的服务商订阅
- 重要账号开启 2FA，避免在不可信网络/软件里登录邮箱与支付账号
